SYSTEM OVERVIEW

HanaYuki v7.0.0 adalah asisten AI canggih yang beroperasi pada jaringan Telegram dengan fokus utama pada keamanan data dan privasi pengguna.

PERINGATAN SISTEM: Bot ini tidak berafiliasi dengan Telegram Inc. Semua interaksi tunduk pada kebijakan privasi ini.

DATA COLLECTION

HanaYuki mengumpulkan data berikut untuk operasional sistem:

  • Telegram User ID (unik)
  • Username Telegram (opsional)
  • Bahasa preferensi sistem
  • Log interaksi (30 hari maks)
  • Pengaturan kustomisasi

DATA YANG TIDAK DIKUMPULKAN: Password, data finansial, informasi sensitif lainnya.

DATA USAGE

Data digunakan secara eksklusif untuk:

  • Optimisasi pengalaman pengguna
  • Personalisasi respons AI
  • Analisis performa sistem
  • Deteksi & pencegahan abuse
  • Pengembangan fitur baru

DATA STORAGE

DATA TYPE RETENTION ENCRYPTION
User Data Active Account AES-256
Interaction Logs 30 Days AES-256
System Logs 7 Days AES-256

USER RIGHTS

Pengguna memiliki hak berikut:

  • Akses ke data pribadi
  • Koreksi data tidak akurat
  • Penghapusan data permanen
  • Pembatasan pemrosesan
  • Penarikan persetujuan
/delete_mydata

Gunakan command di atas untuk menghapus semua data Anda dari sistem.

SECURITY MEASURES

Langkah keamanan yang diterapkan:

  • Enkripsi end-to-end
  • Access control ketat
  • Regular security audit
  • Real-time monitoring
  • Automated threat detection

DATA SHARING

Data TIDAK dibagikan dengan pihak ketiga kecuali:

  • Dengan persetujuan pengguna
  • Kepatuhan hukum yang berlaku
  • Perlindungan hak & keamanan
  • Transfer bisnis (dengan notifikasi)

POLICY UPDATES

Kebijakan ini dapat diperbarui. Pengguna akan diberitahu melalui:

  • Notifikasi via bot
  • Update timestamp
  • Posting di channel resmi

Penggunaan berkelanjutan setelah update berarti penerimaan perubahan.
